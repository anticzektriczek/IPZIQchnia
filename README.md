Głównym zadaniem projektu jest opracowanie i implementacja aplikacji pełniącej funkcje
inteligentnej książki kucharskiej. Jej głównym zadaniem byłoby takie dobieranie przepisów aby
wykorzystać te składniki którymi dysponuje użytkownik.
Dzięki aplikacji będziemy mogli dodawać i wyszukiwać przepisy na podstawie różnych kryteriów,
np. dostępnych składników, sortowanie listy wg wartości kalorycznej lub filtrowanie np. potraw
mięsnych, jarskich czy bezglutenowych.
Zespół ma dowolność w wyborze technologii
